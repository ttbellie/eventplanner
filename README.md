# Event Planner App

## Description
This is an Android Personal Event Planner app developed for SIT305 Task 4.1P.

## Features
- Add event
- View event list
- Update event
- Delete event
- Room database
- Bottom navigation
- Date and time picker
- Input validation

## Tools and Technologies
- Kotlin
- Android Studio
- Room Database
- Jetpack Navigation
- Material Components

## How to Run
1. Open the project in Android Studio
2. Sync Gradle
3. Run on emulator or Android device
